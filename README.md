# icce-cpp

Personal solutions to the exercises from the **C/C++ course** by the Department of Computing Science at the University of Groningen (RUG), created by dr. Frank B. Brokken and Jurjen Bokma.

Working through the material self-study style — not following the course schedule or attending lectures.

## About the Course

The course covers C++ (and by extension nearly all of C), with a strong emphasis on software design principles and understanding *why* code is written the way it is. Topics include the C++26 standard, template concepts, and modules.
